# 星(施工中……)

将AV画质的pdf转成word的工具，尽可能保持格式不变。

## 功能

-   自动旋转矫正

-   表格识别

-   目录识别

-   文字识别

-   图片识别

## 依赖

python 3.6  
tesseract-ocr  
poppler-0.68.0(好像不用)  

```
pip install tqdm numpy scipy scikit-learn pdf2image pytesseract opencv-python cloudpickle
```

都是硬编码的所以自己改代码吧。
